# Amazon AppFlow (amazon-appflow)
Amazon AppFlow is a fully managed integration service that enables you to securely transfer data between SaaS applications like Salesforce, SAP, Zendesk, Slack, and ServiceNow, and AWS services like Amazon S3 and Amazon Redshift, in just a few clicks.

## APIs

### Amazon AppFlow API
The Amazon AppFlow API enables programmatic creation and management of data flows between SaaS applications and AWS services, including configuring connectors, flow definitions, triggers, and data mappings.

**Human URL:** [https://aws.amazon.com/appflow/](https://aws.amazon.com/appflow/)

## Features

| Name | Description |
|------|-------------|
| No-Code Data Flows | Create data flows between SaaS applications and AWS services without writing code. |
| Scheduled Transfers | Schedule data flows to run at configurable intervals using cron or rate expressions. |
| Event-Triggered Flows | Trigger data flows in response to business events from connected SaaS applications. |
| On-Demand Execution | Manually trigger data flows for ad-hoc data transfers. |
| Data Transformations | Apply field mapping, filtering, masking, merging, and arithmetic transformations during transfer. |
| Incremental Data Pull | Transfer only new or changed records since the last flow run using datetime-based incremental pulls. |
| Custom Connectors | Register custom Lambda-backed connectors for proprietary or unsupported data sources. |
| Data Catalog Integration | Automatically catalog transferred data in AWS Glue Data Catalog for discoverability. |
| KMS Encryption | Encrypt data in transit and at rest using AWS KMS customer-managed keys. |
| Private Link Support | Transfer data privately over AWS PrivateLink without traversing the public internet. |

## Use Cases

| Name | Description |
|------|-------------|
| Customer 360 | Consolidate marketing, sales, and support data from Salesforce, Marketo, and ServiceNow into Amazon Redshift or S3 for unified customer analytics. |
| ML Data Preparation | Ingest SaaS data into Amazon SageMaker Data Wrangler for machine learning feature engineering and model training. |
| Data Lake Ingestion | Load SaaS application data into Amazon S3 data lakes for centralized analytics and reporting. |
| Real-Time Analytics | Transfer Salesforce data in near real-time to Amazon Redshift for operational dashboards and BI reporting. |
| Cross-Application Automation | Automate workflows by triggering data flows when records are created or updated in SaaS applications. |
| Data Compliance | Securely extract and archive regulated data from SaaS applications into encrypted AWS storage for compliance auditing. |

## Integrations

| Name | Description |
|------|-------------|
| Salesforce | Bidirectional data transfer between Salesforce CRM and AWS services. |
| SAP | Transfer data from SAP ERP systems via SAPOData connector. |
| ServiceNow | Sync ServiceNow ITSM data to AWS for analytics and reporting. |
| Slack | Export Slack workspace data to Amazon S3. |
| Zendesk | Transfer Zendesk customer support tickets and data to AWS. |
| Marketo | Import Marketo marketing automation data into Amazon Redshift or S3. |
| Google Analytics | Export Google Analytics data to Amazon S3 for custom analytics pipelines. |
| Amazon S3 | Use Amazon S3 as both source and destination for AppFlow data flows. |
| Amazon Redshift | Load SaaS data directly into Amazon Redshift for SQL analytics. |
| Amazon EventBridge | Send AppFlow data to Amazon EventBridge for event-driven architectures. |
| AWS Glue | Catalog AppFlow output data automatically in AWS Glue Data Catalog. |
| Amazon SageMaker | Feed SaaS data into SageMaker Data Wrangler for ML preparation. |
