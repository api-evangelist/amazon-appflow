# Amazon AppFlow

Amazon AppFlow is a fully managed integration service that enables you to securely transfer data between SaaS applications like Salesforce, SAP, Zendesk, Slack, and ServiceNow, and AWS services like Amazon S3 and Amazon Redshift, in just a few clicks.

## APIs

- **Amazon AppFlow API** - The Amazon AppFlow API enables programmatic creation and management of data flows between SaaS applications and AWS services, including configuring connectors, flow definitions, triggers, and data mappings.
